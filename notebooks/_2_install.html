Install gdsfactory#

    1. Install klayout to visualize the GDS files that you create.

    1. Install python 3.7, 3.8, 3.9 or 3.10 I recommend to install python with mamba (faster conda alternative), which makes easier to install plugins such as meep.

    1. Install gdsfactory

    • 3.1 Users can pip install gdsfactory for MacOs and Linux, and for Windows mamba install gdspy -y; pip install gdsfactory

    • 3.2 For gdsfactory+plugins pip install gdsfactory[full] for MacOs and Linux, and for Windows mamba install gdspy -y; pip install gdsfactory[full]. Note that some plugins need to be installed separately (see section below)

    • 3.3 Developers, can git clone it from GitHub and make install (linux and MacOs) or ./install.bat (windows)

Mamba is a faster alternative to conda, for conda you can replace mamba install gdspy with conda install -c conda-forge gdspy -y; pip install gdsfactory

Notice that the latest versions of gdsfactory are no longer compatible with python3.6 and that gdsfactory==2.4.9 was the last version supported by python 3.6. Pip will select this version for py 3.6 environments, which is fundamentally and structurally different to the latest version.

Update gdsfactory#

  1. Users can pip install gdsfactory --upgrade

  2. Developers can git pull on the repository you downloaded and installed on your computer.

Install gdsfactory plugins#

pip install gdsfactory[full] will install most plugins. Some plugins need to be installed separately:

  • meep mamba install meep works for MacOS, linux and windows WSL.

  • sax optimizers, requires jax and jaxlib works for MacOS, linux and windows WSL.

  • sipann pip install gdsfactory[sipann]
